以下都是基于有相同的样式属性
1、相同样式：
	后引入的样式优先级高，但不建议出现。如两个设置div的css，则后面的生效
2、内部样式与外部样式(css文件)与内联样式(标签内)
	内联> 内部与外部     其中内部与外部优先级同样是根据引入顺序决定，后引入的高
3、标签内行间style>id>class>tag>*>继承
4、内联样式 > ID 选择器 > 类选择器 = 属性选择器 = 伪类选择器 > 标签选择器 = 伪元素选择器
其他：
	1、css的!important属性，提升该样式的优先级，会大于style的优先级，不规范。
	
	2、标签+类  >  单类，div.class  >  class

	3、群组选择器与单一选择器优先级相同，即div,p,h1={} = div = p = h1，谁后引入谁优先级高

	4、层次选择器(最好不超过三层)：h1 .class #id {}与#hello div p .class{}。
	权重比较法：计算权重
	如前一个tag 1，class 10，id 100总权重为111；后一个100+1+1+10=112。则后面一个优先级高

	约分比较法：把同级的约掉
	h1与div约分，.class约分，#id约分，则后面一个还剩下p，权重高，则后面优先级高
